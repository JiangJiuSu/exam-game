# 考研游戏系统 — 游戏机制设计文档

> 最后更新: 2026-07-22 | 版本: v1.0

## 一、核心理念

将考研学习过程游戏化，通过学习时长系统、连续签到倍率、目标学校进度条等机制，培养用户在考研期间按时学习的习惯。

---

## 二、学习时长体系

### 2.1 三种时长概念

| 概念 | 说明 | 可编辑 |
|------|------|--------|
| **预定时长** | 创建任务时设定的计划学习时间（分钟），用于倒计时 | ❌ |
| **真实时长** | 实际花费的学习时间（分钟），管理员可修改 | ✅ 管理员模式 |
| **有效时长** | 最终计入总学习时长的值（分钟），自动计算 | ❌ 自动计算 |

### 2.2 有效时长计算规则

```
真实时长 → 是否超过预定时长？
  ├── 是（超时）→ 有效时长 = 真实时长 × 70% × 倍率
  └── 否（按时）→ 有效时长 = 真实时长 × 倍率
```

| 完成情况 | 有效时长计算 |
|---------|-------------|
| 按时完成（倒计时内） | 真实时长 × 当前倍率 |
| 超时完成 | 真实时长 × 70% × 当前倍率 |
| 逾期补完 | 预定时长 × 70% |

### 2.3 学科分类

英语、数学、政治、专业课，四条独立学习时长线。

- **有效时长**：四科之和，用于目标学校进度条
- **真实时长**：四科之和，反映实际学习投入

## 三、目标学校系统

### 3.1 预设学校梯度

| 梯度 | 所需总学习时长 | 示例 |
|------|--------------|------|
| 保底 | 0小时 | 本校/普通院校 |
| 211 | 5,000小时 | 中等难度 |
| 985 | 15,000小时 | 有难度 |
| 顶尖985 | 30,000小时 | 清北复交 |

### 3.2 规则

- 用户选定目标学校后，主界面显示进度条（基于有效时长）
- 门槛数据预设但允许用户自定义修改
- 可随时切换目标学校

## 四、连续签到倍率

### 4.1 倍率增长规则

| 连续天数 | 倍率 |
|---------|------|
| 第1天 | ×1.1 |
| 第2天 | ×1.2 |
| 第3天 | ×1.3 |
| 第4天 | ×1.4 |
| 第5天及以上 | ×1.5（封顶） |

### 4.2 断签规则

- 断签1天 → **倍率归零**，从×1.1重新开始
- 倍率仅作用于**按时完成**的任务
- 超时完成的任务，倍率仍生效但先扣除30%

### 4.3 签到判定

- 当天完成任意1个任务即算签到成功
- 仅登录不算签到，必须有任务完成记录

## 五、任务机制

### 5.1 任务字段（用户填写）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 任务名称 | 文本 | 是 | 例："背50个单词" |
| 所属学科 | 选择 | 是 | 英语 / 数学 / 政治 / 专业课 |
| 预定时长 | 数字（分钟） | 是 | 计划学习时长，用于倒计时 |
| 任务模式 | 选择 | 是 | ⏱️ 倒计时模式 / 📅 截止日期模式 |
| 备注 | 文本 | 否 | 补充说明 |

### 5.2 任务流程

```
创建任务（待开始）
    ↓ 点击"开始任务"
倒计时运行中（实时显示倒计时）
    ↓ 倒计时内点击"完成"
按时完成 ✅ → 有效时长 = 真实时长 × 倍率
    ↓ 倒计时到期
逾期 ⚠️ → 点击"补完" → 有效时长 = 预定时长 × 70%
```

### 5.3 逾期处理

- 倒计时到期自动标记为"逾期"（红色视觉压力）
- 逾期任务仍可补完（拿70%有效时长）
- 删除已完成/逾期任务 → 自动回退学习时长

### 5.4 管理员操作

| 操作 | 说明 |
|------|------|
| 撤销 | 已完成任务恢复为待开始，回退学习时长 |
| 编辑真实时长 | 修改真实学习时间，自动重算有效时长 |
| 删除 | 删除任务并回退其学习时长 |

## 六、可视化组件

### 6.1 学校进度条（主面板）

- 显示：真实学习 → 有效学习 → 目标
- 百分比 + 进度条动画
- 进度条颜色随百分比变化

### 6.2 学习日历热力图

- 按天着色，类似 GitHub 贡献图
- 颜色深浅表示当日学习时长

### 6.3 四科雷达图

- 英语、数学、政治、专业课各自的有效时长
- 暴露偏科问题

### 6.4 连续签到展示

- 当前连续签到天数
- 当前倍率
- 进度条显示距满倍率还差几天

## 七、成就徽章系统（24个）

| 成就 | 条件 |
|------|------|
| XX入门 | 该科学习10小时 |
| XX进阶 | 该科学习50小时 |
| XX精通 | 该科学习100小时 |
| XX大师 | 该科学习500小时 |

### 7.2 签到里程碑（3个）

| 成就 | 条件 |
|------|------|
| 初心者 | 连续签到7天 |
| 坚持者 | 连续签到30天 |
| 毅力者 | 连续签到100天 |

### 7.3 总学习时长里程碑（4个）

| 成就 | 条件 |
|------|------|
| 起步 | 总学习100小时 |
| 积累 | 总学习500小时 |
| 突破 | 总学习1000小时 |
