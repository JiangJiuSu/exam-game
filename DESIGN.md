# 考研游戏系统 — 游戏机制设计文档

> 最后更新: 2026-07-22

---

## 一、核心理念

将考研学习过程游戏化，通过学习时长系统、连续签到倍率、目标学校进度条等机制，培养用户在考研期间按时学习的习惯。

---

## 二、学习时长体系

### 2.1 核心概念

| 概念 | 说明 |
|------|------|
| **预定时长** | 用户创建任务时填写的计划学习时间（分钟） |
| **有效时长** | 实际获得的学习时长，取决于完成情况 |

### 2.2 有效时长计算规则

| 完成情况 | 有效时长计算 |
|---------|-------------|
| 按时完成（倒计时内） | 有效时长 = 实际花费时间 × 当前倍率 |
| 超时完成 | 有效时长 = 预定时长 × 70%（扣除30%） |
| 逾期补完 | 有效时长 = 预定时长 × 70% |

### 2.3 学科分类

英语、数学、政治、专业课，四条独立学习时长线，总学习时长为四科之和。

## 三、目标学校系统

### 3.1 预设学校梯度

| 梯度 | 所需总学习时长 | 示例 |
|------|--------------|------|
| 保底 | 0小时 | 本校/普通院校 |
| 211 | 5,000小时 | 中等难度 |
| 985 | 15,000小时 | 有难度 |
| 顶尖985 | 30,000小时 | 清北复交 |

### 3.2 规则

- 用户选定目标学校后，主界面显示进度条
- 门槛数据预设但允许用户自定义修改
- 可随时切换目标学校，进度条重新计算

---

## 四、连续签到倍率

### 4.1 倍率增长规则

| 连续天数 | 倍率 |
|---------|------|
| 第1天 | ×1.2 |
| 第2天 | ×1.4 |
| 第3天 | ×1.6 |
| 第4天 | ×1.8 |
| 第5天 | ×2.0 |
| 第6天 | ×2.2 |
| 第7天 | ×2.4 |
| 第8天 | ×2.6 |
| 第9天 | ×2.8 |
| 第10天及以上 | ×3.0（封顶） |

### 4.2 断签规则

- 断签1天 → **倍率归零**，从×1.2重新开始
- 倍率作用于按时完成任务的有效时长

### 4.3 签到判定

- 当天完成任意1个任务即算签到成功
- 仅登录不算签到，必须有任务完成记录

## 五、任务机制

### 5.1 任务字段（用户填写）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 任务名称 | 文本 | 是 | 例："背50个单词"、"做一套数学真题" |
| 所属学科 | 选择 | 是 | 英语 / 数学 / 政治 / 专业课 |
| 预定时长 | 数字（分钟） | 是 | 计划学习时长 |
| 任务模式 | 选择 | 是 | 倒计时模式 / 截止日期模式 |
| 备注 | 文本 | 否 | 补充说明 |

### 5.2 任务流程

```
创建任务（待开始）→ 点击"开始任务" → 倒计时运行中 → 完成/超时
```

### 5.3 超时规则

| 情况 | 有效时长 |
|------|---------|
| 按时完成（倒计时内） | 实际花费时间 × 当前倍率 |
| 超时完成 | 预定时长 × 70% |
| 逾期补完 | 预定时长 × 70% |

### 5.3 逾期任务处理

- 超时任务在列表中标记为**红色逾期状态**，形成视觉压力
- 逾期任务仍可完成（拿70%学分），但会一直显示"逾期"标记
- 用户可手动删除逾期任务（学分归零）

## 六、可视化组件

### 6.1 学校进度条（核心）

- 显示在主界面最醒目位置
- 展示当前总学分 / 目标学校所需学分
- 百分比 + 进度条动画
- 接近目标时进度条变色（如：灰→蓝→绿→金）

### 6.2 学分日历热力图

- 按天着色，类似 GitHub 贡献图
- 颜色深浅表示当日获得学分多少
- 展示学习的连续性和积累痕迹

### 6.3 四科学分分布

- 分别展示英语、数学、政治、专业课各自的学分
- 以雷达图或柱状图形式呈现
- 暴露偏科问题，引导均衡学习

### 6.4 连续签到天数

- 醒目显示当前连续签到天数
- 显示当前倍率
- 制造"不想断签"的心理压力

## 七、成就徽章系统

### 7.1 学科里程碑

| 成就 | 条件 |
|------|------|
| 英语入门 | 英语学分首次达到100 |
| 英语进阶 | 英语学分首次达到500 |
| 英语精通 | 英语学分首次达到1000 |
| 英语大师 | 英语学分首次达到5000 |

（数学、政治、专业课各有同系列徽章）

### 7.2 签到里程碑

| 成就 | 条件 |
|------|------|
| 初心者 | 连续签到7天 |
| 坚持者 | 连续签到30天 |
| 毅力者 | 连续签到100天 |

### 7.3 总学分里程碑

| 成就 | 条件 |
|------|------|
| 起步 | 总学分首次达到1,000 |
| 积累 | 总学分首次达到5,000 |
| 突破 | 总学分首次达到10,000 |
| 登顶 | 总学分首次达到30,000 |

### 7.4 成就规则

- 成就纯展示，无实际学分加成
- 成就一旦获得永久保留
- 成就弹窗配合庆祝动画
