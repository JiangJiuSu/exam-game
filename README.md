# 📚 考研游戏系统

> 把考研学习做成游戏，通过倒计时任务、连续签到倍率、目标学校进度条等机制，培养按时学习的习惯。

---

## 🎮 核心功能

### ⏱️ 学习时长系统

| 时长类型 | 说明 |
|---------|------|
| 预定时长 | 创建任务时设定，用于倒计时 |
| 真实时长 | 实际学习时间（管理员可修改） |
| 有效时长 | 最终计入总时长的值（自动计算） |

**计算规则：**
- 按时完成 → 有效时长 = 真实时长 × 倍率
- 超时完成 → 有效时长 = 真实时长 × 70% × 倍率

### 🔥 连续签到倍率

| 连续天数 | 倍率 |
|---------|------|
| 第1天 | ×1.1 |
| 第2天 | ×1.2 |
| 第3天 | ×1.3 |
| 第4天 | ×1.4 |
| 第5天+ | ×1.5（封顶） |

断签1天 → 倍率归零，从×1.1重新开始。

### 🎯 目标学校进度条

| 梯度 | 所需学习时长 |
|------|------------|
| 普通院校 | 0小时 |
| 211 | 5,000小时 |
| 985 | 15,000小时 |
| 顶尖985 | 30,000小时 |

支持自定义目标，可随时切换。

### 📝 任务管理

- **倒计时模式**：设定时长，点击"开始任务"后倒计时，实时显示剩余时间
- **截止日期模式**：设定截止时间
- **逾期处理**：倒计时到期自动标记，补完扣30%
- **日历回看**：按日期查看历史完成/逾期任务

### 🏆 成就徽章（24个）

- 学科里程碑（每科4个：10h/50h/100h/500h）
- 签到里程碑（7天/30天/100天）
- 总学习时长里程碑（100h/500h/1000h/3000h）

### 🔧 管理员模式

- 撤销已完成任务（回退学习时长）
- 编辑真实学习时长（自动重算有效时长）
- 删除任务自动回退时长

## 🚀 快速开始

### 方式一：下载安装包（推荐）

> 适用于普通用户，开箱即用。

1. 从 [Releases](https://github.com/JiangJiuSu/exam-game/releases) 下载 `考研游戏系统 Setup x.x.x.exe`
2. 双击运行安装包，选择安装目录
3. 安装完成，桌面生成快捷方式
4. 双击快捷方式直接启动

**无需安装 Node.js，无需任何配置，开箱即用。**

### 方式二：源码启动

> 适用于开发者，或需要修改代码的用户。

#### 一键启动

1. 下载项目源码（Code → Download ZIP，解压）
2. 双击 **`启动考研游戏.bat`**（或 `.vbs`）
3. 首次运行会**自动下载安装 Node.js**（约1分钟，仅一次）
4. 之后自动安装依赖、启动应用

> 💡 全程无需手动操作，国内网络可正常使用（自动使用国内镜像）。

#### 启动器说明

| 文件 | 说明 |
|------|------|
| `启动考研游戏.bat` | 有命令行输出，便于排错 |
| `启动考研游戏.vbs` | 无黑窗口，双击直接启动 |
| `debug启动.bat` | 调试版，生成 `debug.log` 日志 |
| `debug启动.vbs` | 调试版（无窗口），生成 `debug.log` 日志 |

#### 开发者手动运行

```bash
# 克隆仓库
git clone https://github.com/JiangJiuSu/exam-game.git
cd exam-game

# 设置国内镜像（可选，加速下载）
npm config set registry https://registry.npmmirror.com

# 安装依赖
npm install

# 启动网页版
npm run dev

# 启动桌面版
npm run electron:dev
```

## 📁 项目结构

```
exam-game/
├── electron/
│   ├── main.js                # Electron 主进程
│   └── preload.js             # IPC 桥接
├── src/
│   ├── stores/                # Pinia 状态管理
│   │   ├── creditStore.js     # 学习时长（真实/有效）
│   │   ├── streakStore.js     # 连续签到倍率
│   │   └── taskStore.js       # 任务 CRUD + 倒计时
│   ├── utils/
│   │   ├── achievements.js    # 成就定义
│   │   ├── dataPersistence.js # 数据持久化/备份
│   │   └── initApp.js         # 应用初始化
│   ├── components/            # UI 组件
│   └── views/                 # 页面
├── release/                   # 构建输出（安装包）
├── 启动考研游戏.bat            # Windows 启动器
├── 启动考研游戏.vbs            # Windows 启动器（无窗口）
├── debug启动.bat              # 调试启动器
├── debug启动.vbs              # 调试启动器（无窗口）

├── DESIGN.md                  # 游戏机制设计文档

└── README.md                  # 本文件
```
